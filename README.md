The make file will help you to runn tests on the following data structures:

- Array,
- Dll,
- Stack
- Deque

To compile the array run"
```
make array
```

To run the Dll run:
```
make dll
```

To run the Stack run:
```
make stack
```

To run the Deque run:
```
make deque
```

```make clean``` will remove all o files
